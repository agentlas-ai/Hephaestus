# Code of Conduct

This project exists so people can build and share working agents. Keep
discussion here on that.

## Expected

- Disagree about the code, the architecture, or a design decision as much
  as you want — that's normal and welcome.
- Assume good faith. Ask "why did you build it this way" before assuming
  "this is broken/bad."
- Keep issues and PRs about the thing being discussed. No off-topic
  arguments, no piling on a contributor rather than their patch.

## Not okay

- Personal attacks, harassment, or discriminatory remarks toward anyone
  participating in issues, PRs, or Discussions.
- Posting others' private information without consent.
- Deliberately derailing a thread or spamming links/self-promotion
  unrelated to the discussion.

## Enforcement

A maintainer may edit, lock, or remove content that violates this, and can
block repeat offenders from the repo. If you want to report something
privately rather than in a public thread, open a private security advisory
via GitHub (see [SECURITY.md](SECURITY.md)) and note it's a conduct report,
not a vulnerability — a maintainer will pick it up from there.

This applies to Issues, Pull Requests, and Discussions in this repository.
